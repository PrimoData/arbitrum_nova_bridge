Arbitrum Nova Bridge


Background

Arbitum is a layer 2 Ethereum scaling techonology built by Offchain Labs. Arbitrum One is its first blockchain and one of the more popular ETH L2s. However, Arbitrum recently launched its second blockchain, Nova, and made it avaliable to the public on August 9th, 2022. The purpose of Nova is to have faster and cheaper transactions than Ethereum and Arbitrum One, with a focus on gaming and social platforms (i.e. Reddit). Users can bridge tokens from Ethereum to Arbitrum Nova using the Arbitrum bridge.

Purpose

The purpose of this dashboard & analysis is to understand user adoption/growth of Arbitrum Nova by looking at Ethereum -> Nova bridging activity.

Key Findings
    • ~80% of bridged amounts are <$100, so Nova is primarily used for small transactions where the low transaction costs don't eat away at the assets held.
    • 95%+ of users have only bridged once, so there's opportunity to increase repeat usage.
    • Usage is generally increasing, however, is drowned out by 1 whale who transferred >$440k on August 10th, which accounts for 55% of total $USD value.
Methodology
    • Data pulled directly from the Flipside API for this query and updated daily.
    • Dashboard created using Microsoft Power BI.
    • Data pulled from Flipside's Ethereum table's "ethereum.core.ez_token_transfers" and "ethereum.core.ez_eth_transfers" where the addresses the tokens were sent to were "0xc4448b71118c9071bcb9734a0eac55d18a153949" and "0xc840838bc438d73c16c2f8b22d2ce3669963cd48", Arbitrum Nova's bridging addresses.
    • ETH converted to $USD using the average daily USDC swap rate on Ethereum using the Flipside table "ethereum.core.ez_dex_swaps".
